## What it is
Ollama is a tool that helps you run an AI “chat brain” (LLM) on your own computer/server.

## Why it matters
- You can use AI without sending your data to the internet.
- You can reduce API cost (less paid cloud usage).
- It can run on VPS or laptop (depends on power).

## Key features
- Download and run models locally
- Simple commands to run and chat
- Can be used by other apps as a local AI service
