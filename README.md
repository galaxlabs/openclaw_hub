# OpenClaw Hub

Public notes about AI agents, tools, and LinkedIn job-hunt workflows.

## Folders
- 00_inbox: quick dumps (later sort)
- 01_tools: one tool per file
- 02_agents: agent ideas, designs, workflows
- 03_news: short news notes (with source + date)
- 04_linkedin_posts: post drafts
- 05_job_hunt: saved jobs + drafts
- 06_tutorials: your longer guides

## Writing rules
- Always add Date (YYYY-MM-DD)
- Always add Source (link)
- Always add 3 bullets: What it is / Why it matters / How I will use it
